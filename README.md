# pathfinding_visualizer

This pathfinder uses the grid provided in settings.py and finds a path to the ending node.

Grid Notation:
<ul>
<li># : Wall Node</li>
<li> p : Path Node </li>
<li> s : Start Node </li> 
<li> e : End Node </li> 
</ul>

To change the speed of the pathfinding, change the speed variable in settings.py.
